# 포켓몬 2세대 전투 시뮬레이터 연습

이 프로젝트는 포켓몬스터 2세대(금/은/크리스탈)의 전투 시스템을 Python 백엔드와 Vanilla JS 프론트엔드로 구현해 보는 실습 프로젝트입니다.

## 🚀 시작하기

이 프로젝트는 현대적인 Python 패키지 매니저인 `uv`를 사용합니다.

### 1. 필수 조건
- [uv](https://github.com/astral-sh/uv)가 설치되어 있어야 합니다.

### 2. 라이브러리 설치 및 환경 구축
`pyproject.toml`에 정의된 의존성을 설치하고 가상환경을 생성합니다.
```bash
uv sync
```

### 3. 백엔드 서버 실행
- **일반 모드**:
  ```bash
  uv run main.py
  ```
- **개발 모드 (자동 재시작)**:
  ```bash
  uv run run.py
  ```
서버는 기본적으로 `http://localhost:8000`에서 실행됩니다.

### 4. 프론트엔드 접속
웹 브라우저로 `index.html` 파일을 열거나, 로컬 서버를 통해 접속합니다.
(현재 백엔드에서 `/static` 경로 등으로 정적 파일을 서빙하도록 설정하거나, 직접 파일을 열어 실행할 수 있습니다.)

## 🛠 주요 기능
- **2세대 스텟 시스템**: DV/Stat Exp를 제외한 종족값 기반 능력치 계산.
- **데미지 공식**: 자속 보정(STAB) 및 2세대 타입 상성표 적용.
- **실시간 전투 로그**: 타이핑 효과가 적용된 전투 메시지 출력.
- **애니메이션**: HP 바 변화 및 데미지 피격 효과(예정).

## 📂 파일 구조
- `main.py`: FastAPI 기반의 전투 API 서버.
- `pokemon.py`: 포켓몬 클래스 및 데미지 계산 로직.
- `starters.json`: 스타팅 포켓몬 데이터 (치코리타, 브케인, 리아코).
- `index.html` / `style.css` / `script.js`: 웹 프론트엔드 인터페이스.
- `pokemon_gen2_battle_stats.md`: 참고용 2세대 전투 시스템 조사 문서.
