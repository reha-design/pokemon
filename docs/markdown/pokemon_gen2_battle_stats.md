# 포켓몬스터 2세대(금/은/크리스탈) 전투 시스템 구현 가이드

2세대 포켓몬 전투 시스템을 구현하기 위해 필요한 핵심 능력치(Stats)와 메커니즘을 정리한 문서입니다.

## 1. 핵심 능력치 (Base Stats)

2세대에서 포켓몬의 능력치는 6가지로 구분됩니다. 1세대와의 가장 큰 차이점은 '특수(Special)' 능력치가 '특수공격(Special Attack)'과 '특수방어(Special Defense)'로 분리되었다는 점입니다.

- **HP (Hit Points)**: 포켓몬의 체력
- **공격 (Attack)**: 물리 기술의 위력에 영향을 미침
- **방어 (Defense)**: 물리 기술의 피해를 줄임
- **특수공격 (Special Attack)**: 특수 기술의 위력에 영향을 미침
- **특수방어 (Special Defense)**: 특수 기술의 피해를 줄임
- **스피드 (Speed)**: 선공 여부를 결정 (수치를 직접 비교하며, 동속일 경우 50% 확률로 결정)

---

## 2. 능력치 계산 공식 (단순화 버전)

구현의 편의를 위해 개체값(DV)과 노력치(Stat Exp)를 제외하고, 포켓몬의 **종족값(Base Stats)**과 **레벨(Level)**만을 사용하여 최종 능력치를 계산합니다.

- **HP**:
  `Floor((종족값 * 2) * 레벨 / 100) + 레벨 + 10`
- **그 외 능력치 (공격, 방어, 특공, 특방, 스피드)**:
  `Floor((종족값 * 2) * 레벨 / 100) + 5`

> [!NOTE]
> 실제 게임에서는 DV와 Stat Exp가 포함되지만, 연습용 구현에서는 위 공식만으로도 충분히 전투 밸런스를 맞출 수 있습니다.

---

## 4. 타입 시스템 (Types)

2세대에서 추가된 타입을 포함하여 총 17개 타입이 존재합니다.

- **기존 타입**: 노말, 불꽃, 물, 풀, 전기, 얼음, 격투, 독, 땅, 비행, 에스퍼, 벌레, 바위, 고스트, 드래곤
- **추가 타입**: **강철(Steel), 악(Dark)**
- **물리/특수 분류**: 기술의 물리/특수 여부는 기술별이 아닌 **타입별**로 정해집니다.

| 분류 | 해당 타입 |
| :--- | :--- |
| **물리 (Physical)** | 노말, 격투, 비행, 독, 땅, 바위, 벌레, 고스트, 강철 |
| **특수 (Special)** | 불꽃, 물, 풀, 전기, 에스퍼, 얼음, 드래곤, 악 |

---

## 5. 전투 중 변수 (In-Battle Modifiers)

### 5.1 랭크 보정 (Stat Stages)
- 공격, 방어, 특공, 특방, 스피드 능력치는 -6단계부터 +6단계까지 변화할 수 있습니다.
- **보정치**:
  - `+단계`: (2 + n) / 2 배 (최대 4배)
  - `-단계`: 2 / (2 + n) 배 (최소 0.25배)

### 5.2 명중률 및 회피율
- 기본 100%에서 시작하며, 별도의 랭크 보정(-6 ~ +6)을 받습니다.

### 5.3 상태 이상 (Status Conditions)
- **째려보기/꼬리흔들기**: 상대 방어 1단계 하락
- **울음소리**: 상대 공격 1단계 하락
- *기타 상태 이상(마비, 화상 등)은 현재 로직상 데이터 정의는 되어 있으나 실제 효과는 순차 구현 중입니다.*

---

## 6. 데미지 계산 (Damage Formula)

간소화된 2세대 데미지 계산 공식입니다.

`Damage = (((((2 * Level / 5 + 2) * Attack * Power / Defense) / 50) + 2) * Modifier)`

- **Modifier**:
  - **자속 보정 (STAB)**: 포켓몬 타입과 기술 타입 일치 시 1.5배
  - **상성 보정**: 0.25배 ~ 4배
  - **급소 (Critical)**: 데미지 계산 시 레벨 수치를 2배로 계산하며, 공격자와 방어자의 모든 랭크 보정을 무시합니다.
  - **난수**: 217 ~ 255 사이의 값을 255로 나눈 값 (약 0.85 ~ 1.0)
  - **도구 보정**: 2세대는 지닌 물건(Held Items) 시스템이 처음 도입되었습니다. (예: 목탄 1.1배)

---

## 7. 구현 시 고려사항 (Implementation Tips)

1. **도구(Held Items)**: 나무열매(HP 회복), 박하열매(상태이상 회복) 등 전투 중 자동으로 사용되는 아이템 로직이 필요합니다.
2. **친밀도(Happiness)**: 은혜갚기(Return)와 화풀이(Frustration) 기술의 위력 계산에 사용됩니다.
3. **날씨**: 비바라기, 쾌청, 모래바람 등 날씨 효과에 따른 대미지 증감 및 효과 변화를 고려해야 합니다.
